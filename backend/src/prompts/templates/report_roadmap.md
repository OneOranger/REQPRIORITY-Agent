# 路线图推理任务

请基于以下按季度分组的需求数据，为每个阶段生成 AI 推理说明。

## 路线图数据
```json
$roadmap
```

## 任务：为每个季度生成推理说明

请输出如下 JSON 格式（数组）：
```json
[
  {
    "period": "Q3 2026",
    "reasoning": "本季度聚焦原因和核心目标，要具体说明为什么这些需求要在此阶段完成"
  },
  {
    "period": "Q4 2026",
    "reasoning": "..."
  }
]
```

要求：
- reasoning 要说明每阶段的战略意图，而非简单罗列需求
- 要体现阶段间的递进关系
- 每条 reasoning 50-100 字
- period 字段必须与输入完全一致
