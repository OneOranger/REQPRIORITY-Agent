## 任务一：生成决策报告内容

请输出如下 JSON 格式：
```json
{
  "decisionBasis": [
    "决策依据1，要有数据和逻辑支撑",
    "决策依据2",
    "决策依据3",
    "决策依据4"
  ],
  "riskWarnings": [
    "风险提醒1，具体说明影响和建议",
    "风险提醒2",
    "风险提醒3"
  ]
}
```

要求：
- decisionBasis：4-5条，每条说明当前优先级安排背后的业务逻辑
- riskWarnings：3-4条，具体指出风险点和应对建议
- 不要只是重复数字，要有实质性的判断
